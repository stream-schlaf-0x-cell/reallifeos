## 🎮 GAME MECHANICS

### Resource System

| Resource | Symbol | Generated By | Use Case |
|----------|--------|--------------|----------|
| **Experience (XP)** | ⭐ | Quest Completion | Level progression |
| **Gold** | 💛 | Socratic, Architect | Attack costs |
| **Mana** | 💙 | Bard, Monk, others | Magic/abilities |
| **Movement Points (MP)** | 💚 | Acrobat, Architect | Map exploration |
| **Skill Points (SP)** | ⚡ | Level-ups (+150 per level) | Skill unlocking |

### Progression System

**Level-Up Formula:**
```
XP Required = 100 × 1.5^(level - 1)
Level 1 → 100 XP
Level 2 → 150 XP
Level 3 → 225 XP
(exponential growth)
```

**Level-Up Rewards:**
- +150 Skill Points
- Audio feedback (4-note chord)
- Toast notification
- Log entry

### Skill Tree (5 × 4 Matrix)

Each of the 5 paths has 4 tiers:
1. **Basis (Foundation)** - Tier 1, 0-100 SP cost
2. **Schwelle (Threshold)** - Tier 2, 100-150 SP cost  
3. **Quantensprung (Quantum Leap)** - Tier 3, 200-300 SP cost
4. **Meisterschaft (Mastery)** - Tier 4, 500-800 SP cost

**Skill Attributes:**
- `id`: Unique identifier
- `name`: German descriptive name
- `desc`: Flavor text explaining mechanical benefit
- `icon`: Icon name for visual representation
- `cost`: SP required to unlock
- `req`: Array of prerequisite skill IDs
- `tier`: 1-4 numbering
- `unlocked`: Runtime state flag

### Quest System (Guild Bounties)

5 Daily Quest Types mapped to paths:

**Sokratiker - Die Sokratische Schmiede**
- Description: 30 Min Kavita-Textarbeit or KI-gestützte Unterrichtsplanung

**Barde - Kanalisierung des Flows**

- Resources: +40 Mana
- Color: Purple 🟣

**Mönch - Innere Ausrichtung**
- Description: 20 Min Zazen, Qi Gong, Visualization

- Resources: +50 Mana
- Color: Emerald 🟢

**Akrobat - Physische Resilienz**
- Description: Gym, Laufen, Flow Arts/Jongling

- Color: Red 🔴

**Architekt - Wartung des Motors**
- Description: System-Revision (Craft, Dify, Inbox Zero)

- Color: Blue 🔵

**Custom Quests:**
- Players can create unlimited custom quests
- Define name, path, XP value, description
- Generate own resource allocation
- Stored in localStorage

### Points of Interest (POI) Map Mechanic

World Map generates 19-hex grid with procedural "Archipel des Geistes" POIs:

**POI Types & Effects:**

```
🏯 Kloster (Monastery)
   └─ Effect: +2 Mana per quest completed near this POI
   └─ Name: Tempel der Stille, Zazen-Pavillon, etc.

🎓 Akademie (Academy)
   └─ Effect: +3 Gold per quest near this POI
   └─ Name: Sokratische Akademie, Platos Schatten, etc.

💪 Gym
   └─ Effect: +3 MP per quest near this POI
   └─ Name: Eisenhalle, Dojo der Beharrlichkeit, etc.

🎵 Studio
   └─ Effect: +1 Mana & +1 Gold per quest
   └─ Name: Klanglabor, Ableton-Arkaden, etc.

🖥️ Server-Farm
   └─ Effect: +2 Gold & +1 MP per quest
   └─ Name: Docker Core, Dify-Rechenzentrum, etc.

⚔️ Wilds (Wildnis)
   └─ Effect: Random combat encounter (Ambush)
   └─ Name: Nebel der Täuschung, Schlucht der Prokrastination, etc.

🌀 Nexus
   └─ Effect: +1 all resources per quest (unique, central)
   └─ Name: Zentrum des Geistes, Quelle der Klarheit, etc.
```

**POI Mechanics:**
- Undiscovered tiles cost **10 MP** to uncover
- Upon discovery, permanent resource bonus applied to subsequent quests
- Each POI has randomized names from theme-appropriate word bank
- Resource bonuses stack multiplicatively
- Map regenerates procedurally on new save

### Combat System

**Battle Arena:** Daily randomized boss encounters

**Boss Properties:**
```javascript
{
  id: "boss_1",
  name: "Dämon der Prokrastination",
  maxHp: 200,
  type: "Mental",
  color: "text-purple-400" // Tailwind color
}
```

**Available Attack Types:**

1. **Basic Attack** - Damage only
   - Cost varies by path (5-25 Mana)
   - Damage: 45-80 HP
   
2. **Critical Hit** - 2.5x damage
   - Cost: Mixed (Mana + Gold)
   - Requires specific skill unlock
   - Visual: Yellow glow, "💥" indicator

3. **Shield** - Damage reduction
   - Blocks damage for N turns
   - Cost: Mana-heavy
   - Requires skill unlock
   - Visual: Blue glow, "🛡️" indicator

4. **Heal/Utility** - Support actions
   - Can damage boss while self-healing
   - Variable costs
   - Requires skill progression

**Combat Example:**
```
Player: "808-Donner" attack
Cost: 20 Mana
Boss HP: 200 → 120
Floating damage: "-80" (red text, animated up)
Boss animation: shake + flash
ComboLog: "808-Donner against Prokrastination: 80 damage"
```

**Combat Flow:**
1. Action selection UI shows available attacks
2. Resource check (Mana >= cost && Gold >= cost)
3. Damage calculation (normal/crit multiplier)
4. Apply damage to current boss
5. Animate floating number + boss reaction
6. Log entry
7. Check defeat condition
8. Next combat turn ready

**Boss Defeat:**
- All boss HP → 0
- New random boss selected
- Reset HP to maxHp
- Toast: "Boss besiegt! Neuer Boss erscheint!"
- Gold/XP reward (optional skill-based multiplier)

## 🗺️ WORLD MAP SYSTEM

### Hexagon Grid Navigation

**Coordinate System:** Axial hex coordinates (q, r)
- Pointy-top orientation
- 45px hex size
- Procedural generation with concentric rings

**Player Actions on Map:**
1. **Discover Tile** (10 MP cost)
   - Reveals type, name, POI bonus
   - Enables later quest-with-bonus gameplay
   
2. **Visit POI** (via quests)
   - Activate bonus effects
   - Can stack multiple POI bonuses
   
3. **Combat Boss** (map-specific bosses)
   - Each tile can have optional mapBoss
   - Defeat grants special rewards
   - Visually distinguished on map

**UI Controls:**
- Pan buttons: ← ↑ ↓ → adjust viewBox
- Zoom buttons: + − adjust scale
- Discovered/Total counter: Shows progress
- Active bosses display

### Map Generation Algorithm

```javascript
generateMap(radius = 4)
1. Create concentric hex rings from center
2. Distribute POI types with weighted probabilities:
   - Nexus: 1 (always center)
   - Monasteries: 3-6
   - Academies: 3-6
   - Gyms: 2-5
   - Studios: 2-4
   - Servers: 2-4
   - Wilds: 8-18 (fill)
3. Assign random names from theme lists
4. Optionally place map bosses on ~15% of tiles
5. Initialize all as non-discovered
6. Return tile array with all properties
```

## 💾 STATE MANAGEMENT

### useGameState Hook Architecture

**Central store in `src/hooks/useGameState.js`**

**Persisted State Structure:**
```javascript
{
  // Progression
  level: number,
  xp: number,
  skillPoints: number,
  
  // Skills (flattened array)
  skills: [{
    id: string,
    name: string,
    desc: string,
    icon: string,
    cost: number,
    req: string[],
    tier: number,
    unlocked: boolean,
    isCustom?: boolean
  }],
  
  // Combat
  currentBoss: BossObject,
  bossHp: number,
  playerShield: number,
  shieldTurnsLeft: number,
  combatLog: string[],
  defeatedBosses: BossObject[],
  
  // Resources
  movementPoints: number,
  gold: number,
  mana: number,
  
  // Quests
  customQuests: [{
    id: string,
    name: string,
    path: string,
    xp: number,
    completed: boolean,
    completedAt: ISO8601string
  }],
  
  // Map
  mapData: {
    tiles: [{
      q: number, r: number,
      type: POItype,
      name: string,
      discovered: boolean,
      mapBoss?: BossObject
    }]
  },
  poiBonuses: { manaRegen, goldRegen, moveRegen },
  
  // Logging
  log: [{
    timestamp: ISO8601,
    time: timestring,
    type: LOGTYPE,
    message: string
  }],
  day: date string,
  
  // Transient (not persisted)
  damageEvents: [],
  lastDamageAmount: number,
  lastDamageType: "hit" | "crit" | "heal" | "shield"
}
```

### Key State Functions

**addXp(amount)**
- Increments xp
- Checks level threshold: `100 × 1.5^(level-1)`
- Multi-level possible in 1 call
- Awards +150 SP per level
- Plays audio, shows toast

**handleQuestComplete(quest)**
- Gets base rewards from path
- Calculates POI bonuses
- Updates resources
- Calls addXp
- Logs entry
- Shows toast

**executeAttack(action)**
- Validates resources
- Calculates damage
- Applies to boss HP
- Triggers animation
- Updates combat log
- Checks defeat

**unlockSkill(skillId)**
- Validates SP >= cost
- Checks prerequisites met
- Sets skill.unlocked = true
- Deducts SP
- Plays sound
- Logs entry

**uncoverTile(tileIndex)**
- Checks 10 MP >= cost
- Sets tile.discovered = true
- Recalculates POI bonuses
- Deducts MP
- Triggers animation
- Persists immediately

**addCustomSkill(path, tier, skillData)**
- Generates unique ID
- Sets isCustom = true
- Appends to skills array
- Makes immediately available
- Shows toast

**addCustomQuest(name, path, xp, desc)**
- Generates unique ID
- Initializes completed = false
- Stores in customQuests
- Persists immediately

**claimTat(questId)** (Quest Completion)
- Marks custom quest complete
- Triggers full reward flow
- Logs activity

## 🛠️ TECHNICAL ARCHITECTURE

### Tech Stack
- **Framework:** React 19.2.4
- **Build:** Vite 8.0.4
- **Styling:** TailwindCSS 4.2.2 + PostCSS
- **Language:** JavaScript (JSX)
- **Runtime:** Browser (ES Modules)
- **Storage:** localStorage API

### Component Hierarchy

```
<App>
  └─ <Layout>
      ├─ <Header>
      │  └─ XP bar, resource display, dev toggle
      ├─ <Navigation>
      │  └─ Tab selector (desktop: horiz, mobile: bottom)
      └─ {activeTab === "tree" && <SkillTree>}
         {activeTab === "quests" && <Quests>}
         {activeTab === "battle" && <BattleArena>}
         {activeTab === "map" && <WorldMap>}
      
      {devMode && <DevToolsPanel>}
      {showLog && <ActivityLogView>}
      {showAddQuest && <AddQuestModal>}
```

### Data Flow

```
User Action
  ↓
Event Handler in Component
  ↓
Dispatch state update via hook
  ↓
useGameState merges new state
  ↓
useEffect persists to localStorage
  ↓
Component re-renders with new state
  ↓
Visual update + animations
```

### Styling System

**TailwindCSS with custom extensions:**
- Dark mode by default: `bg-slate-950 text-slate-200`
- Path-specific gradients in `PATH_COLORS`
- Responsive modifiers: `md:`, `lg:` breakpoints
- Custom animations: `animate-float-dmg`, `animate-shake`
- Glow effects: `shadow-[0_0_20px_rgba(...)]`

**Theme Colors:**
```css
Socratic:  Amber    (#F59E0B)
Bard:      Purple   (#A855F7)
Monk:      Emerald  (#10B981)
Acrobat:   Red      (#EF4444)
Architect: Blue     (#3B82F6)
```

### Audio System

**Web Audio API Synthesizer** in `audioEngine.js`

- **playHitSound()** → [65.41, 130.81] Hz (1s)
- **playLevelUpSound()** → [261.63, 329.63, 392, 493.88] (3s) 
- **playUnlockSound()** → [440, 659.25] Hz (2s)

Features:
- Sine wave oscillators
- LFO modulation (0.1-2 Hz) on detune
- Exponential envelope (attack → decay)
- Master gain control

### Animation System

**CSS Animations:**
```css
@keyframes float-damage { /* damage numbers rise + fade */ }
@keyframes shake { /* boss recoil */ }
@keyframes flash-hit { /* damage feedback */ }
@keyframes crit-flash { /* critical multiplier */ }
@keyframes stripes { /* XP bar animation */ }
```

**React State-Driven:**
- useEffect detects lastDamageAmount change
- Triggers animation classNames
- Clears after duration
- Smooth state transitions

### Canvas Particle Background

**particleEngine.js:**
- 70 particles spawned randomly
- Drift with periodic direction changes
- Wraps at canvas edges
- 60 FPS via requestAnimationFrame
- Draws under all UI (opacity: 40%)


```
4. Return to app, mark quest complete
5. See resources increase, view level progress
6. Unlock new skill with accumulated SP
7. Optional: Map exploration to find POI bonuses

9. Close app (state auto-saves)
```

## 🔧 EXTENSIBILITY GUIDE

### To Add New Path (6th Dimension)
1. Add color to `PATH_COLORS` in `constants.js`
2. Create new path object in `skillTreeData.js` with 4 tiers
3. Add path ID to `getResourceRewards()` function
4. Create quest template in `quests.json`
5. Add path emoji + config to quest components
6. Update navigation tabs if needed
7. Add icon + styling

### To Add New Combat Action
1. Add entry to `attacks.json` with cost/damage/type
2. Include `requiresSkill` if gated
3. In `useGameState.js`, handle action type in `executeAttack()`
4. Create animation style in component
5. Add floating damage visual for new type
6. Test resource validation

### To Add New POI Type
1. Add to `POI_TABLE` in `useGameState.js` with bonus config
2. Add to `POI_TYPES` weight in `mapGenerator.js`
3. Add random names to `POI_NAMES` in `mapGenerator.js`
4. Create SVG icon + styling
5. Add appearance radius glow color in `WorldMap.jsx`
6. Update bonus recalc logic if complex
